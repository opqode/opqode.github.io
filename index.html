Features

  • Content and Style Separation

    RinohType accepts content represented using a semantic markup language such as reStructuredText (optionally using Sphinx) or DocBook.

  • Simple Styling

    Change any aspect of your document's style by creating a custom style sheet. RinohType's style sheets are very similar to, but more powerful than the web's Cascading Style sheets (CSS).

  • Document Templates

    Choose from one of the templates included with RinohType and optionally customize it to your needs. Or create a custom template from scratch.

  • Automatic Float Placement

    RinohType automatically handles the placement of floating document elements such as figures and tables, so you don't have to worry about it.

More Features

Unicode

Built on Unicode, RinohType is ready to support multiple languages.

Fonts

Handles OpenType, TrueType and Type1 fonts with support for advanced typographic features such as kerning and ligatures.

Embed Images

Embed PDF, PNG and JPEG images as-is, preserving transparency and color profiles.

Lightweight

RinohType is a pure-Python library and has few dependencies, so it is very easy to deploy.
